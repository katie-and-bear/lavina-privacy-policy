隱私權政策

最後更新日期：2025年1月

1. 資料收集

我們重視您的隱私權。LAVINA Pinterest 創作助手工具收集的資料類型包括：

  • Pinterest 帳戶的基本資訊（用戶名稱、頭像）
  • 圖板和 Pin 的公開資訊
  • 使用工具時的操作記錄

2. 資料使用

收集的資料僅用於：

  • 提供 Pinterest 管理和創作服務
  • 改善工具功能和使用者體驗
  • 提供技術支援

3. 資料安全

我們採取以下措施保護您的資料：

  • 使用 Pinterest 官方 API，確保資料傳輸安全
  • 本地處理資料，不在伺服器端儲存敏感資訊
  • 定期更新安全措施

4. 第三方服務

本工具整合 Pinterest API 服務，請同時參閱 Pinterest 隱私權政策。
